Java Code Explanation - Pet.java

Code:

package Pet;

public class Pet {
    private static int nextId = 2;
    private final int id;
    private String name;
    private int happiness;
    private int hunger;
    private int energy;

    private static final int MAX_STAT = 100;
    private static final int MIN_STAT = 0;

    public Pet(String name, int happiness, int hunger, int energy) {
        this.id = getNextId();
        this.name = name;
        setHappiness(happiness);
        setHunger(hunger);
        setEnergy(energy);
    }

    public static int getNextId() {
        return nextId++;
    }

    public int getHappiness() {
        return happiness;
    }

    private void setHappiness(int happiness) {
        this.happiness = Math.max(MIN_STAT, Math.min(MAX_STAT, happiness));
        printHappinessStatus();
    }

    public int getHunger() {
        return hunger;
    }

    private void setHunger(int hunger) {
        this.hunger = Math.max(MIN_STAT, Math.min(MAX_STAT, hunger));
        printHungerStatus();
    }

    public int getEnergy() {
        return energy;
    }

    private void setEnergy(int energy) {
        this.energy = Math.max(MIN_STAT, Math.min(MAX_STAT, energy));
        printEnergyStatus();
    }

    public void updateStatus() {
        setHappiness(happiness - 3);
        setHunger(hunger + 5);
        setEnergy(energy - 2);
    }

    public void feed() {
        setHappiness(happiness + 10);
        setHunger(hunger - 20);
    }

    public void play() {
        setHappiness(happiness + 15);
        setHunger(hunger + 10);
        setEnergy(energy - 15);
    }

    public void rest() {
        setHunger(hunger + 5);
        setEnergy(energy + 20);
    }

    public String getStatus() {
        return "Pet: " + name + " | Happiness: " + happiness + " | Hunger: " + hunger + " | Energy: " + energy;
    }

    private void printHappinessStatus() {
        if (happiness == 100) System.out.println("Happiness Status: Very happy");
        else if (happiness > 75) System.out.println("Happiness Status: Happy");
        else if (happiness > 50) System.out.println("Happiness Status: Little happy");
        else if (happiness > 25) System.out.println("Happiness Status: Little sad");
        else if (happiness > 0) System.out.println("Happiness Status: Sad");
        else System.out.println("Happiness Status: Very sad");
    }

    private void printHungerStatus() {
        if (hunger == 100) System.out.println("Hunger Status: Starving");
        else if (hunger > 75) System.out.println("Hunger Status: Very hungry");
        else if (hunger > 50) System.out.println("Hunger Status: Hungry");
        else if (hunger > 25) System.out.println("Hunger Status: Little hungry");
        else if (hunger > 0) System.out.println("Hunger Status: Fine");
        else System.out.println("Hunger Status: Full");
    }

    private void printEnergyStatus() {
        if (energy == 100) System.out.println("Energy Status: Full of energy");
        else if (energy > 75) System.out.println("Energy Status: Very energized");
        else if (energy > 50) System.out.println("Energy Status: Energized");
        else if (energy > 25) System.out.println("Energy Status: Little tired");
        else if (energy > 0) System.out.println("Energy Status: Out of energy");
    }
}

Questions and Answers

Q1: What does `nextId` do?

A1: `nextId` is a static variable that increments for each new pet, ensuring unique IDs.

Q2: Why is `id` declared `final`?

A2: `final` makes `id` unchangeable once assigned, ensuring each pet’s ID remains unique and constant.

Q3: What is the purpose of `MAX_STAT` and `MIN_STAT`?

A3: These constants set the bounds for happiness, hunger, and energy values.

Q4: Why do `setHappiness`, `setHunger`, and `setEnergy` use `Math.max` and `Math.min`?

A4: To keep stat values within 0 and 100, preventing values outside the allowed range.

Q5: What does `updateStatus` simulate?

A5: It reduces happiness, increases hunger, and decreases energy, simulating changes over time.

Q6: How does `feed()` affect the pet’s stats?

A6: It increases happiness by 10 and decreases hunger by 20, representing the pet being fed.

Q7: What happens in `play()`?

A7: Happiness increases by 15, hunger by 10, and energy decreases by 15 as the pet plays.

Q8: What is the purpose of `printHappinessStatus()`?

A8: It provides a description of the pet's happiness level based on its current value.

Q9: Why does `getNextId()` use `nextId++`?

A9: It returns the current `nextId` and then increments it for the next pet.

Q10: Why are happiness, hunger, and energy private?

A10: To control how these attributes are modified, preventing direct changes and allowing validation through methods.
